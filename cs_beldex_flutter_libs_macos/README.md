# `cs_beldex_flutter_libs_macos`
macOS implementation of `cs_beldex_flutter_libs` for the
[`cs_beldex` package](https://pub.dev/packages/cs_beldex).  Refer to the main
monorepo at https://github.com/MogamboPuri/cs_beldex for more context.

## Usage
This package is [endorsed](https://flutter.dev/to/endorsed-federated-plugin), which means you can simply
`flutter pub add cs_beldex_flutter_libs` and this package will be automatically
included in your app as needed without any explicit reference in `pubspec.yaml`.  
You only need to add this package to your `pubspec.yaml` if you want to import
and use any of its APIs directly.

## Optional
This package and its bundled binaries are not required to use `cs_beldex`, but
is is provided as a quick and convenient way to start coding with the library.  
Refer to `cs_beldex`'s documentation for more information on building the
libraries from source and use this package as a template or example for bundling
the required binaries.
