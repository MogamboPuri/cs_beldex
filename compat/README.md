# `compat`

Compatibility layer to ease migration from 
[`flutter_libbeldex`](https://github.com/MogamboPuri/cs_beldex/flutter_libbeldex) to 
[`cs_beldex`](https://pub.dev/packages/cs_beldex).  Uses code copied from 
`flutter_libbeldex`.  Refer to the main monorepo at 
https://github.com/cypherstack/cs_monero for more context.
